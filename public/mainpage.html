Objective

The primary objective of this system is to enhance the existing framework of the MLALAF Scholarship Program. This scholarship attendance monitoring system aims to streamline data retrieval processes and improve overall efficiency.

Project Overview

The Scholarship Attendance Monitoring System is a proposed solution for the course "System Integration and Architecture 2," under the guidance of Mrs. Arlene Garcia. This system is designed to address the current challenges within the Mlalaf Scholarship Program by organizing data more effectively and providing a more efficient approach to attendance monitoring.
